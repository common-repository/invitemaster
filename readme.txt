Технологии вирусного маркетинга для увеличения посещаемости вашего блога

== Description ==

Allows people to recommend any page to their friends with title & short page description (teaser).

It's integrated with http://invitemaster.ru online service, so WP user does not need place any html code into their post or php templates, 
but simply adjust tell-a-friend button parameters in plugin.

Plugin is available only in Russian!

== Installation ==

1. Upload "invitemaster.zip" through the "Plugins" - "Add New" - "Upload"
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust plugin parameters
